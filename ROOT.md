# ROOT

A placeholder to indicate the project ROOT.

There are three ways to find the ROOT directory of this project:

- The `ROOT.md` file itself, which is located at the root of the project.
- The `VERSION` file, which is also located at the root of the project and contains the current version of the project.
- Use `git rev-parse --show-toplevel` command in the terminal, which will return the absolute path to the root directory of the Git repository.
