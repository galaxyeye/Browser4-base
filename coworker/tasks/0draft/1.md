# Build AriaSnapshot from DOMTreeNodeEx

Implement the method `render` in `AriaSnapshotRenderer` to build an `AriaSnapshot` from a given `DOMTreeNodeEx`.
The `AriaSnapshot` should capture the relevant ARIA attributes and their values from the DOM node, as well as any necessary
information about the node's role and state.

```kotlin
ai.platon.browser4.driver.chrome.dom.model.AriaSnapshotRenderer.render(ai.platon.browser4.driver.chrome.dom.model.DOMTreeNodeEx)
```

Create comprehensive e2e tests to verify that the `render` method correctly extracts ARIA attributes and constructs the `AriaSnapshot` as expected.
