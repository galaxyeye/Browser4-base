# Improve Scheduled Coworker Task

There are multiple scheduled tasks in the coworker system without a unified approach to handling them.
This leads to inconsistencies in how tasks are created, managed, and executed.

设计一个完整、可直接运行的 PowerShell Scheduler 方案。设计目标：

- 支持 多个独立任务
- 并发执行
- 统一调度
- 日志与状态记录
- 只需要 一个 Windows Task Scheduler 定时触发
- 可通过配置文件配置任务是否开启，任务的执行频率等参数

必须在单独的进程中定期执行 coworker / draft refinement / task source monitor 等任务。

## 改进点

1. **统一调度器**：设计一个统一的调度器来管理所有任务，避免每个任务都需要单独的脚本和定时器。
2. **配置文件**：使用一个配置文件来定义所有任务的参数，如是否启用、执行频率等，简化任务管理。
