# 实现高阶 browser4-cli 方法

## Advanced Usage

```bash
browser4-cli load https://example.com/form -expire 1h
browser4-cli submit https://www.amazon.com/dp/B08PP5MSVB -deadline 2026-2-24T23:59:59Z
browser4-cli submit --seed-file=seeds.txt
browser4-cli scrape https://www.amazon.com/dp/B08PP5MSVB --selector=".product-title" --attribute="textContent" --output=title.txt

browser4-cli agent summarize https://www.amazon.com/dp/B08PP5MSVB --prompt "Tell me about something about this page" --output=summary.txt
browser4-cli agent act "scroll to the middle"
browser4-cli agent extract "product name, price, ratings" --output=product.json

browser4-cli agent run 'visit x.com, search for @galaxyeye8, and summarize the latest 5 posts'

browser4-cli agent run '
Go to https://www.amazon.com/dp/B08PP5MSVB

After browser launch:
  - clear browser cookies
  - go to https://www.amazon.com/
  - wait for 5 seconds
  - click the first product link
After page load: scroll to the middle.

Summarize the product.
Extract: product name, price, ratings.
Find all links containing /dp/.
'

browser4-cli close
```
